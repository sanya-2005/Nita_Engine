как происходит система евентов сейчас
есть MessageQueue. там хранятся евенты и евент хандлеры
мессейдж куеуе выполняет обработку сообщений надлежащими хандлерами
евент состоит из евент даты и типа евента
евент дата - содержит информацию для обработки этого евента
евент хандлер - содержит метод Invoke, который выполняет обработку евента

проблема:
взятие данных для евента
евент куеуе содержится в классе окна. окно доступно только из движка. еще возникла проблема, что для обработки евента могут понадобится данные из нескольких мест. например: ResizeEvent - евент масштабирования окна. для обработчика нужен OSWindow, и !!! - RenderEngine. Render Engine пока невозможно получить из места, где формируется event
